# 语言类学科试题生成器
一个用来生成语文或者英语试题的系统
# 我该如何做？
你只需要将文言文或者单词列表，按指定格式排版好。然后运行这个软件
等一分钟左右，这个软件会自动将试题生成并保存
你接下来要做的就是把试题复制进Microsoft Word或者 Adobe Acrobat、Incopy中排版。
